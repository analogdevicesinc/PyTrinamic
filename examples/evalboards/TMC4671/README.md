## Wiring

### Motor

| Cable Color | TMC6100 |
|-------------|---------|
| Black       | U       |
| Red         | V       |
| Yellow      | W       |

### ABN Encoder

| Cable Color | TMC4671 |
|-------------|---------|
| Red         | +5V     |
| Black       | GND     |
| Withe       | ENC1_A  |
| Green       | ENC1_B  |
| Yellow      | ENC1_N  |

### Digital Hall

| Cable Color | TMC4671 |
|-------------|---------|
| Red         | +5V     |
| Black       | GND     |
| Blue        | HALL_UX |
| Green       | HALL_V  |
| Withe       | HALL_WY |
